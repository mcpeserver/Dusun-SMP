# Pusat Dukungan (Support Guide)

Halo warga Dusun SMP! Jika kamu mengalami kendala teknis saat mengakses website, kesulitan menyalin alamat IP, atau tidak bisa terhubung ke server Minecraft, kami siap membantumu!

Berikut adalah beberapa saluran dukungan resmi yang bisa kamu gunakan:

---

## 💬 1. Dukungan Komunitas (Rekomendasi)

Untuk respon tercepat, bergabunglah dengan grup diskusi aktif kami:
* **WhatsApp Group**: Masuk ke grup WhatsApp warga melalui tombol yang ada di website untuk berdiskusi langsung secara kasual dengan ratusan warga lainnya.
* **Discord Server**: Masuk ke server Discord Dusun SMP. Kami memiliki channel `#tanya-jawab` dan `#bantuan-teknis` tempat kamu bisa bertanya langsung kepada admin, moderator, maupun pemain berpengalaman lainnya.

---

## 🛠️ 2. Melaporkan Masalah Teknis Website

Jika kamu menemukan error tampilan atau kendala khusus pada landing page ini:
* Silakan buka [GitHub Issues](https://github.com/username/dusun-smp/issues) dan laporkan menggunakan template **Bug Report**.
* Developer kami aktif meninjau feedback di platform tersebut untuk terus melakukan optimalisasi performa website.

---

## 👑 3. Pertanyaan Mengenai Akun Minecraft

Jika akun kamu tersangkut, mengalami masalah lag parah, kehilangan item akibat bug game, atau ingin melaporkan tindakan pemain lain yang melanggar aturan:
* Silakan buka tiket bantuan resmi (ticket system) di dalam Discord Dusun SMP.
* Tim staf / pengelola server akan membantu memulihkan data atau menyelesaikan perselisihan secara adil.
