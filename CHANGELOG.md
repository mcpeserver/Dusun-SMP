# Changelog (Catatan Perubahan)

Semua perubahan penting pada proyek website resmi **Dusun SMP** akan didokumentasikan di file ini.

Proyek ini mengikuti prinsip penomoran versi [Semantic Versioning](https://semver.org/).

---

## [1.0.0] - 2026-07-06

### ✨ Ditambahkan (Added)
- **Desain Landing Page Utama**: Membangun landing page premium satu halaman (single-page landing) dengan tema *Survival Village Indonesia*.
- **Integrasi Developer API**: Integrasi bar informasi developer dinamis di bagian atas header dan footer menggunakan data API GitHub Raw.
- **Pengecekan Status Server**: Widget pendeteksi status online server Minecraft asli menggunakan API publik mcsrvstat.us secara aman dan real-time.
- **Salin IP & Port Cepat**: Fitur copy-to-clipboard sekali klik untuk IP server, Port Java, dan Port Bedrock dengan indikator sukses visual.
- **Panduan Bermain Dual-Platform**: Langkah-langkah bergabung mendetail untuk pemain Java Edition (PC) dan Bedrock/PE Edition (Mobile/Android).
- **Aturan Resmi Terpusat**: Section aturan bermain (rules) modern menggunakan penomoran otomatis dan ikon visual pendukung dari Lucide React.
- **Konfigurasi `src/config/site.ts`**: Skema konfigurasi terpusat untuk memudahkan pengurus server mengubah alamat IP, port, deskripsi, sosial media, fitur, dan aturan tanpa mengubah kode inti.
- **Dokumentasi Terlengkap**: Menyediakan file standar open source (`LICENSE`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, `SUPPORT.md`, `.editorconfig`, `.gitattributes`, `.prettierrc`, issue & PR templates).
