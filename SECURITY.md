# Kebijakan Keamanan (Security Policy)

Kami sangat berkomitmen untuk menjaga keamanan data dan kenyamanan para pengguna website serta warga server Dusun SMP. 

Jika kamu menemukan kerentanan keamanan (vulnerability) pada website ini atau infrastruktur server yang terkait, mohon kerja samanya untuk **tidak mempublikasikannya secara terbuka** terlebih dahulu demi mencegah eksploitasi pihak yang tidak bertanggung jawab.

---

## 🛡️ Cara Melaporkan Celah Keamanan

Hubungi tim pengelola atau developer kami secara pribadi melalui saluran berikut:

1. **WhatsApp Pengelola**: Hubungi nomor kontak resmi yang tercantum di bagian footer website (kontak developer).
2. **Discord PM (Private Message)**: Hubungi moderator atau staf admin berkedudukan tinggi di server Discord Dusun SMP.
3. **Email**: Kirimkan deskripsi celah keamanan beserta langkah-langkah eksploitasi (proof of concept) ke email penanggung jawab.

Kami akan meninjau laporan kamu dalam kurun waktu **24-48 jam** dan segera merilis perbaikan patch keamanan sesegera mungkin.

Terima kasih atas kontribusimu dalam menjaga Dusun SMP tetap aman untuk semua warga!
